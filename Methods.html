Gene expression and splicing quantification

Our quantification pipeline is based on the nf-core/rnaseq pipeline that we have modified to include the following quantification methods:

  1. Total gene expression: HISAT2 + featureCounts
  2. Splice junction usage: Leafcutter
  3. Exon expression: DEXSeq
  4. Transcript usage: Salmon
  5. Trancriptional event usage: txrevise + Salmon

The pipeline is available from GitHub .
